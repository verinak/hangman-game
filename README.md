# hangman-game

A simple Hangman game built with **vanilla HTML, CSS, and JavaScript (EC5)**.  
<!-- View Github repo at [verinak/hangman-game](https://github.com/verinak/hangman-game) -->
<!-- View demo [here](https://verinak.github.io/hangman-game/) -->

<br />

## Features
- Random word generation from multiple categories
- Hangman visual changes with each guess
- Win/Lose screen displayed after game is over
- "Play Again" button to restart game
- Responsive design that works on all screen sizes
